# AI写歌知识宝典

一个单页面的 AI 音乐创作提示词百科，专为 **Suno** 和 **Solo** 用户打造。  
无需安装任何软件，浏览器打开即用。

## 在线访问

👉 **[点击这里直接使用](https://huangyaling658-creator.github.io/ai-songwriting-bible/)**

> 如果链接暂时无法访问，说明 GitHub Pages 还在部署中，请稍等几分钟后刷新。

## 本地使用

1. 点击页面右上角绿色的 **Code** 按钮，选择 **Download ZIP**
2. 解压后，双击 `index.html` 文件
3. 浏览器会自动打开，即可使用全部功能

也可以用 Git 克隆：

```bash
git clone https://github.com/huangyaling658-creator/ai-songwriting-bible.git
open ai-songwriting-bible/index.html
```

## 包含哪些内容

| 模块 | 说明 |
|------|------|
| **乐器** | 70+ 种乐器的中英文提示词，按键盘、弦乐、管乐、打击、民族、电子六大类分组 |
| **风格与流派** | 40+ 种音乐风格卡片，含流派介绍、关键词标签、代表艺人，部分风格可在线试听 |
| **人声演唱** | 嗓音类型、演唱技巧、和声层次、音色效果四大分类，每条附带 Suno 可用的示例句 |
| **Suno 技巧** | AI 音乐模型的运作原理、Suno 各功能的使用方法、常见问题与解法 |
| **结构** | 歌曲段落结构、氛围情绪、速度节奏、特殊提示词 |

## 核心功能

- **Prompt 生成器**（右下角 + 按钮）：从任意模块点击 + 号收集关键词，自动拼成自然英文 Prompt，一键复制到 Suno
- **分类框架**：生成的 Prompt 按"风格 → 乐器 → 氛围 → 人声 → 段落 → 速度 → 特殊"的顺序组织，结构清晰
- **中英对照**：所有提示词同时显示中文释义和英文关键词
- **在线试听**：风格模块内置 iTunes 预览音频，点击即可试听风格示范
- **历史记录**：最近 10 条生成的 Prompt 自动保存在浏览器本地，刷新不丢失

## 技术说明

- 纯静态单 HTML 文件，无需服务器、无需联网（试听功能除外）
- 数据全部内嵌，无外部依赖
- 历史记录使用浏览器 localStorage 存储
- 支持移动端自适应布局
